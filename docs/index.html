📋 適合度の評価基準

📈 リターン（40%）

  • OK +20%以上: 高リターン
  • OK +10%〜20%: 中リターン
  • 中 0%〜10%: 小リターン
  • NG マイナス: 損失

🎯 勝率（30%）

  • OK 60%以上: 高勝率
  • 中 40%〜60%: 中勝率
  • NG 40%未満: 低勝率

🔢 取引回数（30%）

  • OK 10回以上: 十分な機会
  • 中 5〜9回: 機会あり
  • NG 5回未満: 機会少

📉 最大下落率

  • OK 20%未満: 低リスク
  • 中 20%〜40%: 中リスク
  • NG 40%以上: 高リスク
